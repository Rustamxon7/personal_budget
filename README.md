## Getting Started

To get a local copy up and running follow these simple example steps.

git clone ?

cd Personal-Budget

### Prerequisites
Ruby 3.0.1
Rails 6.1.4.4
PostgresSQL

### Setup
Install gems with:
```
bundle install
```
Setup database with:
```
rails db:create
rails db:migrate
```

### Usage
Start server with:

```
rails server
```

Open http://localhost:3000/ in your browser.

### Run tests

Install rspec with:
```
bundle install
```
and
```
rails generate rspec:install
```
run the test with:
```
rspec spec
```
